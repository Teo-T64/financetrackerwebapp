## Features
*  Secure User Authentication via JWT
*  Responsive Dashboard with Tailwind CSS
*  Full CRUD functionality for deleting, filtering, creating and updating information about your income/expenses, categories
*  Visual representation of your spending habits with line-charts and pie-charts using ```recharts```
*  Income and expense data can be downloaded as an .xlsx file and sent to user's e-mail.

---

## Installation 
### Clone the repository
* ```git clone https://github.com/Teo-T64/financetrackerwebapp.git```
* ```cd financetrackerwebapp```
* ```npm install```
* Content of the ```.env``` file is the backend URL, if you have deployed your own make sure to include it in the project.
